# Traditional vs AI-Augmented Development

<div class="mt-4">

### Traditional Approach

```
1. Read requirements
2. Research solutions
3. Write code manually
4. Debug and iterate
5. Write tests
6. Document
```

**Time:** Hours to days per feature

</div>

::right::

<div class="pl-8 mt-4">

### AI-Augmented Approach

```
1. Define context & constraints
2. Collaborate with AI on design
3. Generate code with AI assist
4. Review and refine together
5. AI-assisted testing
6. Auto-generate docs
```

**Time:** Minutes to hours per feature

</div>

<div class="absolute bottom-12 left-12 right-12">

<div class="p-3 bg-green-500/10 rounded-lg text-sm">
<carbon-information class="inline text-green-400" /> The human remains essential - AI amplifies, doesn't replace
</div>

</div>
