# LLM Strengths & Limitations

<div class="mt-4">

### What LLMs Do Well

<div class="space-y-2">
<div><carbon-checkmark class="text-green-400" /> Pattern recognition</div>
<div><carbon-checkmark class="text-green-400" /> Text transformation</div>
<div><carbon-checkmark class="text-green-400" /> Code generation</div>
<div><carbon-checkmark class="text-green-400" /> Explaining concepts</div>
<div><carbon-checkmark class="text-green-400" /> Summarization</div>
<div><carbon-checkmark class="text-green-400" /> Translation (code & language)</div>
<div><carbon-checkmark class="text-green-400" /> Brainstorming</div>
</div>

</div>

::right::

<div class="pl-8 mt-4">

### What LLMs Struggle With

<div class="space-y-2">
<div><carbon-close class="text-red-400" /> Precise math calculations</div>
<div><carbon-close class="text-red-400" /> Real-time information</div>
<div><carbon-close class="text-red-400" /> Counting (sometimes!)</div>
<div><carbon-close class="text-red-400" /> Consistent state across turns</div>
<div><carbon-close class="text-red-400" /> Knowing what they don't know</div>
<div><carbon-close class="text-red-400" /> Following very long instructions</div>
<div><carbon-close class="text-red-400" /> Avoiding hallucinations</div>
</div>

</div>

<div class="absolute bottom-12 left-12 right-12">
<div class="text-sm p-3 bg-red-500/10 rounded-lg">
<carbon-warning class="inline text-red-400" /> Never blindly trust AI output. Always verify critical logic and facts.
</div>
</div>
